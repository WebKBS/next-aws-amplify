## Summary

우선 Vercel 배포 완료후 hosting kr에 도메인 연결하는 방법을 알아보겠습니다.

## Hosting Kr 가입후 도매인 구매

[`Hosting.kr`](https://www.hosting.kr/)에 가입후 도메인을 구매합니다.

> 도메인 구매는 일반 쇼핑하듯이 도메인 검색후 원하는 도메인을 구매하면 됩니다.<br/>
> 도메인 구매 방법은 생략!

도메인 구매후 나의 서비스 - 도메인관리 메뉴로 이동합니다.

<MDXImage
  src="./images/domain-setting.png"
  title="구매한 도메인 리스트"
  width="500px"
/>

구매한 도메인 리스트가 나오는데, 연결하고자 하는 도메인을 클릭합니다.

상단 탭 메뉴에서 네임서버/DNS를 클릭합니다.

네임서버에 보면 기본으로 설정되어 있는 네임서버가 있습니다.

아마 기본으로 설정되어 있는 네임서버는<br/>
ns1.hosting.co.kr<br/>
ns2.hosting.co.kr<br/>
ns3.hosting.co.kr<br/>
ns4.hosting.co.kr<br/>
이렇게 되어 있을것 입니다.

네임서버 오른쪽 끝 연필 모양을 누르고 편집모드로 들어갑니다.

ns1.hosting.co.kr -> ns1.vercel-dns.com<br/>
ns2.hosting.co.kr -> ns2.vercel-dns.com

이처럼 네임서버를 변경해 줍니다.

변경사항을 저장합니다.

hosting.kr에서의 설정은 끝났습니다.

## Vercel Dashboard로 이동.

vercel dashboard에서 상단 Tab 메뉴의 Domains을 클릭합니다.

아래 이미지와 같이 나오는데, Add 버튼을 클릭합니다.

<MDXImage src="./images/vercel-dashboard.png" title="Vercel Dashboard" />

버튼을 클릭하면 현재 배포된 모달창이 나타나는데 원하는 프로젝트롤 Select 후 Continue 버튼을 클릭합니다.

<MDXImage src="./images/add-domain1.png" title="Add Domain" />

다음 Add Domain 설정에서 hosting kr에서 구입한 도메인을 입력합니다.

<MDXImage src="./images/add-domain2.png" title="Add Domain" />

그리고 Add 버튼을 누릅니다.

이제 시간이 조금 걸리지만, 도메인이 연결되면서 https로 접속이 가능해집니다.

참 쉽죠? 😄
